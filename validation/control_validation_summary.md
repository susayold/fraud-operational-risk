# Control Validation Summary

> This project is an educational fraud and operational-risk framework. It combines an observed PCA-transformed fraud benchmark with a separately labelled synthetic controls-testing environment. It is not a production fraud platform, live payment control, AML system or regulatory model.

All **38 UAT**, **18 SIT** and **16 negative tests** executed through reference functions or artifact checks. Rule conditions, boundaries, priority, reason codes, actions, capacity, fallback and overlap are traceable. Synthetic rule precision and recall remain below 100%, and model/rules/anomaly incremental value is reported rather than assumed.
